# About

All my talks, in all their glory, previously made with [RevealJS](https://revealjs.com/) and now made with [SliDesk](https://slidesk.github.io/slidesk-doc/docs/intro/).

# Structure

[`./common`](`./common`) contains all the "framework" code, that is then symlinked in each talk directory.
[`./archives`](`./archives`) are where I put my talks to rest.

Talk contents is in Markdown format, with bits of HTML/CSS here and there for (moderately) complex slides.

I also try to add a PDF format for archival considerations. You can use them liberally as long as you ping me about it (I like to see how my work is quoted ! ^^).

# Requirements

Node.js >= v22
